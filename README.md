## Setup Instructions

### Prerequisites
- Node.js (v16 or higher)
- npm (Node Package Manager)

### Backend Setup
1. Navigate to the twc-test-api
```bash
   cd twc-test-api
```

2. In the terminal type the following command
```bash
   npm install
   node app.mjs
```

3. Navigate to the twc-test-web
```bash
   cd twc-test-web
```

4. In the terminal type the following command
```bash
   npm install
   npm run dev
```
